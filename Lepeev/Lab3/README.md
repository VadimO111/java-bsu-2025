# Lab 3: Nonogram

### Зачем нужны Java и докер?
**Java** — база для надежного бэкенда, который потом не придется переписывать каждые полгода. **Spring Boot**  позволяет не изобретать велосипед и писать миллион строк бесполезного кода, а просто пользоваться готовыми решениями, чтобы связать фронт, базу и бизнес-логику.
**Докер** нужен для решения вечной проблемы «у меня на компе всё работало». Дико удобно, что одной командой в терминале можно поднять сразу все сервисы (базу, бэк) и не париться с установкой окружения.
